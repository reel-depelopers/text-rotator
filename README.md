# Vertical Rotator Block

テキストを縦方向にアニメーションで切り替える、モダンなWordPress用Gutenbergブロックプラグインです。

## 特徴

- モダンな開発手法: block.json をベースにした最新のWordPress標準に準拠。
- バニラJS: 特別なビルド設定（npm等）なしで動作。
- レイアウト保持: テキストの長さに応じて周囲のレイアウトが崩れない「場所取り」方式を採用。
- カスタマイズ性: エディター上でテキストカラーやフォントサイズ、揃え方向、単語リストを変更可能。
- ホバー停止機能: ユーザーがマウスを乗せるとアニメーションが一時停止する、心地よいUX。

## インストール方法

1. GitHubからリポジトリをダウンロード（または git clone）。
2. wp-content/plugins/ ディレクトリに vertical-rotator フォルダとして配置。
3. WordPress管理画面の「プラグイン」メニューから有効化。

## 使い方

1. 固定ページまたは投稿のエディターを開きます。
2. ブロック追加ボタン（+）から "Vertical Rotator" を検索して追加。
3. 右側のサイドバー設定パネルで以下をカスタマイズ：
   - 単語リスト: カンマ区切りで入力（例: 超える,創る,未来）。
   - テキスト揃え: 左/中央/右 の中から選択。
   - カラー設定: 好きな色に変更。

## 技術詳細

- apiVersion: 3
- render_callback によるサーバーサイドレンダリング。
- get_block_wrapper_attributes でTypographyサポートを最適化。
- CSSによるスムーズなフェード＆スライドアニメーション。
